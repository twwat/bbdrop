# Keyboard Shortcuts

## Gallery Table

| Shortcut | Action |
|---|---|
| Delete | Remove selected galleries from queue |
| Ctrl+C | Copy BBCode for selected galleries |
| F2 | Rename selected gallery |
| Home | Jump to first gallery in current tab |
| End | Jump to last gallery in current tab |
| Enter | Save inline edit and move to next row |
| Ctrl+Click | Select multiple galleries |
| Shift+Click | Select a range of galleries |

## Tabs

| Shortcut | Action |
|---|---|
| Ctrl+T | Create a new tab |
| Ctrl+W | Close current tab (Main tab cannot be closed) |
| Ctrl+Tab | Switch to next tab |
| Ctrl+Shift+Tab | Switch to previous tab |
| Double-click tab | Rename tab |

## Application

| Shortcut | Action |
|---|---|
| F1 | Open Help dialog |
| Ctrl+, | Open Settings |
| Ctrl+. | Show keyboard shortcuts |
| Ctrl+R | Toggle right panel visibility |

## Upload Workers Table

| Shortcut | Action |
|---|---|
| Enter | Open host configuration dialog |

## Drag and Drop

- **Folders** — Drag image folders into the gallery table to add them to the queue
- **Archives** — Drag ZIP, RAR, or 7Z files to extract and queue their contents
