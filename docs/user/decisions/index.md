# Architecture Decision Records

Architecture Decision Records (ADRs) are lightweight documents that capture
significant architectural decisions along with their context and consequences.
Each record describes a choice the project faced, what was decided, and the
tradeoffs that follow.

New ADRs should use the `NNN-slug.md` naming convention and follow
[Michael Nygard's ADR template](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).
