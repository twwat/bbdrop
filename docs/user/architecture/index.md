# Architecture diagrams

This section documents BBDrop's architecture using the
[C4 Model](https://c4model.com/) (Simon Brown). The C4 Model provides a set of
zoom levels --- from a high-level system context down to individual components
--- so that each diagram tells a clear story at the right level of detail.
